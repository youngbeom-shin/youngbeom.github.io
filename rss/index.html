又到了年底这个时间点，打算开始整理一下这一年的一些过往，回头想想这一年都发生了什么，经历了哪些事情。算是给这一年的时间做一个阶段性的总结。

这一年总结下来应该是自毕业到现在最艰难的一年，感觉也是我们成长最多的一年。

关于生活


今年生活最大的变化就是我们迎来了我们的女儿七七，整个一年从七七出生开始一分为二，前半年大体都是轻松的，后半年是我们经历起伏最多的半年。

七七出生前一天正好是一个周末，我自己一个人从看房签约再到搬家一共就用了一天时间，当天晚上八点多搬完家后在回去的路上，老婆说已经开始规律宫缩了，我们可能得准备行李去住院了，这时距离预产期还有十天左右的时间。

就这样在母亲节这一天我们办理了入院手续，第二天14:06，我们的女儿出生了，我还记得把她抱到病床边的时候她一直在哭，声音很细，听到我说：“女儿是爸爸呀！” 她就一下停住不哭了，这句话也是在胎教的时候经常说的一句话，第一次感受到了新生命带来的惊喜，我和女儿有了第一次的互动。

从七七出生到现在年底这半年时间是我和我爱人经历过所有的时间里最困难的一段时期，知道照顾新生儿很辛苦很累，但真的经历过后才会更有实感。

从住院期间开始因为照顾宝宝不得不频繁夜醒，老婆也在经历身体激素变化带来的例如情绪波动，我们都瘦了很多。她的产后体重降到了90斤，我也是在最初的一两周瘦了将近10斤。

后面出院后我们就搬到了新搬的房子，家里迎来了新的成员，之前的房子就住不下了。索性就找了一个大房子，感觉什么都可以放得下了。

老婆的产假休了半年，正好在家这段时间是北京最热的时候，那段时间老婆应该是最难熬的，本来就是因为坐月子不方便开窗开空调，再加上从七七出生开始就因为哺乳没有睡过一天整觉，整个人看上去特别的虚弱，虽然月嫂也一直变着花样的做月子餐，但对她的消耗还是太大了。

在七七出生前我们就想过无数种生活的可能，和孩子的姥姥或奶奶一起生活，两方的妈妈轮流帮忙看孩子，我们也尽可能的想到各种可能出现的问题，想好了各种预案，避免出现家庭问题等等，但最终的情况是，我们的妈妈都放弃帮我们照顾孩子离开了北京，然后我们现在跟育儿嫂一起生活，到现在我们都回到公司上班，只有育儿嫂自己一个人在家带孩子，我们白天只能通过监控看到孩子的情况。

这应该是我们之前预想过的最糟糕的情况，也是我们当时设想过最最不放心的情况，但最终我们却发现这可能是目前为止最好的了，我们到目前为止换了4位阿姨，现在的阿姨是最最负责任的阿姨，也是经验最丰富的。我们还在调侃，没想到因为七七的到来我们过上了一日三餐有阿姨照顾的日子，家务和育儿的工作也是由阿姨接手，宝宝现在跟阿姨相处的很好，我们只有在早晚的时间能多陪陪孩子，所以也格外的珍惜工作之外的时间。

在这段时间身体上的累还是可以克服，但最最让我感到无力的是我们自己的家人，我曾经觉得在我们最困难的时候愿意帮助我们的一定是父母，但父母却在我们最无助的时候选择了离开。

从大概一年前开始为了让孩子奶奶尽快能适应新的生活，真的是做了很多预防工作，每个月不同节日都送上节日问候和比平常更多的红包，还给他们讲各种育儿相关的事情，希望能在轻松互助的环境下照顾好新生儿，她也表示愿意照顾这个新生命。

为了让妈妈适应环境提前接到北京，周末的时候老婆还挺着肚子一起出门打卡家附近的餐厅，一起去公园露营，逛颐和园看车展，真是为了能让妈妈开心做了很多的努力，但最后的结果是孩子的奶奶在孩子出生第七天老婆还没出月子的情况下坚持要回去，理由是身体不适，很难接受这样的理由，当时月嫂还没上户，留下我一个人照顾一个还没满月的婴儿和正在坐月子的老婆。

后面实在是需要人手帮忙，岳母在月嫂已经上户的情况下在家里住了一个半月，期间也是经常会有一些育儿观念上的差异，老婆也会在和岳母相处的过程中感受到更多的压力和干涉，通常育儿嫂和阿姨不会给我们这种困扰。

正常在产假期间应该是尽可能的减轻产妇的各种身心灵上的负担，但是这两段和父母相处的时间却是让我们最费心的时间。两位妈妈都是内心比较敏感的性格，所以我们不但要照顾孩子，还要照顾父母的情绪，一点也轻松不起来。后来索性我们就想着不再去寻求父母的帮助，什么事情都自己来。

最近一次阿姨聊天中她说你们都是独生子女，作为你们的父母还可以全力帮衬你们，我当时都不知道该说什么，可能因为父母的承诺让我们期待了太多。

期间还经历2次月嫂因为家人生病需要照顾临时替换阿姨的经历，真的是一次次的挫折和试炼，每一次刚感觉经历了一段难熬的时间刚要放下心的时候就又要面临新的问题，作为我还好，对于一个孩子的母亲来说真的承受了非常多的压力，那段时间真的是每天都非常焦虑。这种面对太多不确定带来的压力，甚至让我们觉得来自金钱上的压力是最不值得一提的。

其实孩子长大的过程中需要面对的问题还远不止如此，最近七七已经开始能表现出更多的情感，在早上上班送别的时候能从她的眼神中已经看到失落的表情了，之前还是会毫不在意的自己在玩玩具，不知道后面越来越懂事的时候会是什么情形。

也正是这样那样的问题，让我们夫妻之间的关系变得更加的亲密，我们必须做到一起面对这些摆在眼前的问题。我突然理解之前听到的一句话，就是如果夫妻关系没有好到一定程度千万不要尝试要孩子，这绝对是考验夫妻感情的非常重要的时期。

抛开让我们头疼的各种问题还有有非常多的开心幸福的时候，这半年来真的是一天天的看到女儿的成长，从她第一次笑，到她第一次学会翻身，一直到最近每天下班回到家开心的笑，抱在怀里咿咿呀呀的发出 baba mama 的声音，真的是我能体会到最幸福的时刻，是孩子一直在治愈我们，我们也希望我们可以给孩子留下更多美好的瞬间。

我们每天都有很用心的记录她的成长，一有空就会拿起相机记录，希望她能够在我们这个小家庭中幸福快乐的长大。

今年最大的变化就是在家的时间应该是历年之最，少了很多和朋友们的社交活动，少了很多我们两个人的娱乐时间，一有时间就带着七七晒太阳逛公园。

以上应该就是生活上的一些变化了吧。

关于投资理财

今年的储蓄率应该是我们自工作以来最低的一年？我们换到了更大的房子，请了一位住家的阿姨，这个应该是花销的大头，除了这两项之外，孩子的一些其他开销并没有我预想的多。

今年也没有定一个家庭总资产的目标，想着就放轻松一点，如果事事都讲究计划目标，那应该还挺辛苦的。

今年在投资理财上有两个尝试，一个是在今年年初银行利率降低前买了两笔外资银行大额存单，固定利率3.5，提前锁定了5年的利率收益。现在看最新的招商银行5年期定存的利率是1.55。每年都能拿到3.5的收益放在今天还是挺难得的，放到前两年可能都不会放弃流动性投资这种低收益率的产品。

另外一个是给我们的女儿七七开启了一个定投计划，我希望是每个交易日为七七定投纳斯达克100元，这样一直存到她18岁生日。希望她成年的时候有一笔自己的可以自由支配的存款，这笔存款她可以用来作为探索世界的门票，或者其他她认为必要的开销，或者单纯作为自己的小金库，虽然不多，但是也足以作为一些帮助。

这个定投计划如果是坚持18年的话，假设按照年化利率3%计算的话总余额会是59W，如果是年化收益是5%总余额会达到72W。虽然过往收益并不代表未来收益，但单纯看纳斯达克100指数过去18年的年化收益率在16-18%之间。如果在18年前开始这项定投工作，并按照年化收益率16%计算，本金收益和将达到228W。希望这个简单的复利计算能够让更多的人了解到定期储蓄的必要性。

但也是要在风险承受范围内进行投资理财，谁也不知道什么时候再会发生一次特殊事件，比如2000年左右的美股互联网泡沫、08年的次贷危机引发的全球金融危机、还有就是疫情期间美股的数次触发熔断都是非常可怕的，那个时候的最大回撤也很吓人。假设一个人是在金融危机之后开始定投，但是定投了10年左右遇到了疫情期间的熔断，那么也将面临着非常大的本金亏损。

这个定投计划我也是看到一些推特上的爸爸们开始为自己的孩子开启的新的理财计划，我希望这个计划我可以雷打不动的坚持下去。

还有一个是因为对于明年的人民币利率走低的预期增强，在年底的时候买一笔美元理财，标的是中银的一个货币基金，可以理解为美元版本的余额宝，当下虽然利率也在降低，但是最近的预估收益率也可以达到4.2，这是目前人民币里面的R1理财中绝对找不到的产品。

计算了一下我是在7.3的时候买入的，如果按照3%的收益率计算，人民币兑美元汇率升到7左右是不亏也不赚的，如果人民币贬值到7.5左右，那收益率综合起来将会有7%左右。最近世界银行和各大投资机构也都发出了明年人民币汇率的预估，很多的机构预估都是7.5左右。

关于工作

工作上今年没有太大变动，年初的时候接到了一家公司迟来的offer，但当时我已经加入了现在的团队。现在的团队经历了这一年的时间战略清晰了很多，产品线也更加的聚焦了，产品的功能成熟度也好了很多，也在招投标中拿到了大国企的订单。

团队目前已经开发了大模型资产管理平台，智能数据平台，还有代码编辑器插件等产品都在积极迭代中。

对于个人来讲今年整体的工作节奏还是很舒服的，每月一个版本，功能都是提前规划好，设计师出好方案跟踪开发进度并测试上线，这些都很常规。

关于旅行

今年应该是历年来最宅的一年了，主要是担心七七还小，担心经常换环境和作息会让她没有安全感，所以打算等到她再大一点的时候安排旅行计划。

今年就两个旅行地，一个是今年元旦的时候和老婆一起去了武汉跨年，另一个是春节期间去了济州岛。

在武汉的时候虽然整体行程比较紧张，但也是比较难忘的旅行，我们在光谷一起度过了跨年夜，也去了博物馆看了越王勾践剑，老婆挺着肚子还去爬了黄鹤楼，说实话我都没有孕妇能走，那个时候还不是孕晚期所以她的体力都很好。

武汉整体的印象就是很年轻，光谷有特别多的大学和年轻人，感觉消费都比北京旺盛，但当地人也讲也是因为大学生多，所以毕业生的薪资待遇都很卷，很多学生毕业就往一线城市去了，也是不太容易留下人才。

对比北京来说武汉是一个比较新的城市，虽然武汉的历史也很久，但是因为武汉有很多新建开发的商业和住宅，看起来就像是一个整体很新的城市，不像北京除了一些CBD，很多地方都是老楼。

可能也是因为今年整体房地产不景气的原因，遇到了两个印象特别深的普通人，一位是我们从武昌区过长江隧道到汉口时的出租车司机，他说早年在广东开餐馆赚钱后想着陪孩子所以搬来了武汉，买了两套恒大的房子，到现在还没拿到房子就烂尾了，估计钱是拿不回来了，两套房子当时买的时候应该是市值200W，就当是做生意赔钱了，就不继续耗着了。

另一位是我们早上在早点摊吃完热干面散步的时候碰到的一个民工大叔，我们走到一个老旧小区附近时，路边有一个已经停业的报刊亭，上面贴满了广告，除了一堆租房广告之外还有一个律师帮忙打官司催债之类的广告，大叔打电话说自己有6万左右的工资被拖欠了，问对方能不能帮忙讨回，可能是对方觉得金额太小还是难度比较大，说了两句大叔就挂断了电话。我们当天吃的热干面6元一份，这么看6万对比当地的物价真的不是一个小数目。

对于旅行给我们最好的体验应该就是当地的饮食真的很合我们的胃口，当地的排骨藕汤、还有红烧武昌鱼、小龙虾都是非常值得推荐的特色。还和老婆说下次希望找一个合适的时间一定要品尝当季的小龙虾。

关于济州岛的旅行写到了另外一篇博客里：域外杂谈-济州岛旅行见闻

今年的五一十一等长假我们都没有出行，而是留在北京，今年趁着有时间办了韩国的F4签证，因为可以到时候拿着F4长居签证去当地换登陆证（韩国身份证），换国际驾照和办理境外银行卡，但在首尔办理登陆证需要最少2周左右的时间才能出来电子证明，4周左右才能拿到韩国身份证，所以对于没有太多假期的我们来说一直没有合适的机会，直到后面看到去地方出入境比如蔚山一天就可以出来电子证明，拿着电子证明次日就可以回国，在第二次去的时候就可以拿到登陆证用中国驾照换取国际驾照和办理银行卡。

正好孩子的姥爷就在蔚山工作生活，我们也可以当作探亲，更加的方便了。有了国际驾照再去到海外旅行地的时候就可以在当地开车自驾了，毕竟有了孩子我们也没办法总是特种兵式旅行。有了境外银行卡也更方便我们做境外理财投资，算是一举多得。

因为前同事去年移居日本工作之后总是跟我提起在日本的一些见闻，让原本就想去日本旅行的我更加想去看看了，年底的时候就给我们全家搬了日本签证，原本是计划跨年在关西，但是因为行程定的比较晚，加上元旦期间日本上班族开启了9连休，整个关西地区的机酒价格就变得比平常高了很多，带着小孩推着婴儿车也不太好去人挤人的场所，所以虽然签证已经办好了，我们也决定等开春了天气暖和一点的时候挑一个非节假日再启程，一个是机酒价格会便宜近一半，另外就是没有太多人所以旅行体验会好一些，可以不紧不慢的慢慢感受当地的特色文化。

年度照片

24年的跨年我们是在武汉过的，正好有机会去看了湖北省博的越王勾践剑，也顺带了解了那个时期的一些历史。

武汉也是一个快速发展的城市，整个城市在快速的更新迭代中，前景的老房子和背景的新房形成了鲜明的对比，过去老一辈的生活面貌正在一点点消失。

吃完早餐回来的路上看到的打法律援助电话讨债的老汉，说是有6万的工程款没有结清，我们早上吃的一晚热干面6元，对比物价这欠款不算是一笔小数目，看到的小广告中最多的是租房，其次是讨债。

跨年夜前还和孕中期的老婆一起怕了黄鹤楼，实在是担心她的身体，但她还是坚持爬完，不得不佩服这个体力。

光谷的跨年夜很热闹，商场门口的空地举办了好多活动，据说武汉是全世界大学生最多的城市，明显看到的现象就是消费还挺旺盛的。本来以为会有倒计时活动，结果被通知临时取消，好多城市也都取消了倒计时活动，我们也担心人群聚集的越来越多，就早早的回去了。

UX Engineer（UXE）是一个最近我才经常听到的一个岗位名称，因为目前为止国内很少有招聘这个岗位的公司，身边更是没有朋友任职过这一岗位。通过名字来看就是 UX Designer 和 Engineer 的结合，可以说是创意全能型人才。

最近也开始看到国外的公司开始有这个岗位招聘，比如 Google 和 GitLab 都有这一岗位，这是他们的岗位介绍页面 [Google-why Google need ux engineer] [GitLab-UX Engneer]。

Medium上有一篇《What Does a UX Engineer Do, Exactly? A Little Bit of Everything》的文章就介绍了在谷歌从事该岗位的工程师的日常。

那么 UXE 到底是什么？其实这个岗位比较难以描述，因为它需要身兼数职、拥有各种知识背景和专业知识，他像是一个设计师和开发之间的桥梁，要会使用设计语言和编程语言。比如在以下工作场景中咨询 UXE 是最恰当不过的选择了：

  • 是否需要研究一些新技术来确定可行性
  • 是否需要开发一个用于用户测试的功能原型
  • 使用新技术和新平台来构建前端的最佳实践

通过以上的例子我们可以大致了解这个岗位需要处理和完成的工作内容。

UX Engineer 的角色与职责

UXE 是一个交流沟通的桥梁，也需要具备创造性的解决问题的能力。

他是介于设计师和开发者之间的角色，所有通常会有两个不同的视角来思考手里的工作。

作为设计师的角色会需要在设计评审中针对设计原型和方案提出建议，并尽可能的去预测可能出现的设计以及体验问题，这个时候就需要设计师的经验来帮助我们，设计师通常需要具备将用户或者客户的模糊需求转化为可行性方案的能力，并且通过与各相关方的讨论以及用户测试去验证可行性方案，最终输出设计给到开发者。

同时作为开发者的角色中我们又要保证代码的整洁清晰、尽量减少技术债务、合理结构，减少开发成本，并且基于目前最合适的软件工程最佳实践来开展开发工作。

就如前面文章提到的那位在谷歌担任这一职位的作者描述的一样，虽然他可能不会向所有人推荐成为 UX Engineer 的角色，但因为具备了更多的相关知识和同理心，更能轻松的和设计师、研发、项目经理合作，因此这个角色能够更好的在团队中发挥效力，并且更重要的是会成为 HR 眼中的独角兽。

在 GitLab 的 UE 岗位描述中我们可以了解不同职级的 UXE 的工作职责，通常包括以下内容：

  • 改善产品用户体验和界面
  • 开发设计系统组件并完善设计系统
  • 在最小团队支持下，持续不断的迭代产品
  • 了解并分享最新的 UI/UX 技术、工具和模式

成为 UX Engineer 的学习路径

首先要了解这个岗位的要求之后，我们才可以更好的去学习如何成为一个合格的 UXE。

下面的一段是 GitLab 对 UXE 的岗位要求：

  • 精通 HTML、CSS 和 JavaScript 等前端编程语言。
  • 具有使用 Vue.js 框架的专业经验。
  • 具有深厚的设计系统知识、理解和实践。
  • 对视觉设计、交互设计和可用性测试有深入的了解。
  • 有使用 Git 和 Yarn 或 NPM 等包管理器的经验。
  • 具有编写自动化测试的经验（例如 Jest、Karma、Jasmine、Mocha、AVA、tape）。
    对于 senior 的 UXE 的要求还包括：
  • 需要深入了解产品知识
  • 参与设计评审
  • 要通过撰写博客、发表演讲等方式提升影响力
  • 跨部门合作
  • 带领指导团队成员和负责招聘

我们分为几个部分来看首先是设计基础，需要了解学习用户体验设计的基本原则，通常在设计师最早接触 UX 这个岗位时最多推荐的一本书就是经典的《About Face: The Essentials of Interaction Design》，虽然最早是1995年首次出版，但依然是 UX 设计领域中的权威著作之一。里面会介绍一系列的交互设计的基本原则，比如一致性、简洁性、可见性等。这些原则帮助设计师在设计时确保用户能顺利理解并使用界面。

如果你是经验丰富的开发者，那么对于 UX 的学习可能会快很多，这里推荐 tailwind css 的作者发布的一本书 《RefactoringUI》，通过一些例子来快速的提升产品的用户体验。

但在使用的过程中对于一些 css 调整以及自适应布局等基础概念有了实际上手操作的机会。

最近的两份工作经验中，一个是让我更熟悉了代码托管工具和 Git 相关的知识，另一个则是让我真正有了贡献开源项目代码的机会。

目前我们团队有一个开源在 GitHub 上的产品，前端是使用 Vue 和 tailwind 写的，因为项目人员有限并且时间紧张的关系，我们只能先解决有没有的问题，关于好不好用以及好不好看的问题只能排在第二优先级，但作为设计师看到自己的设计实现还原度失真还是很难接受，所以我就动了自己上手修改前端代码的想法。

真的希望国内也能有更多更优秀的 UX Engineer 涌现出来，也希望我能将目前团队开源的这个产品的用户体验真正提升到优秀的水平，也让我通过项目提升更多的开发技能。

什么是 SVG Animation？


SVGs（scalable vector graphics）是可缩放矢量图形文件的缩写，是一种标准的图形文件类型，一种用于描述二维图形的XML标记语言。因为是矢量文件，所以无论是在多大的尺寸上展示效果都是一样的清晰，保证了图像不受尺寸影响，依然能够是最佳的展示效果。

不同于以像素为单位的位图，svg是基于代码，因此 SVG 动画具有可拓展性，后期更容易修改和调整。

Svg 更好的一点是，他的体积非常的小，对于网页来讲更容易渲染和加载，所以在现代网页设计中被广泛使用。

使用 Figma 将同一个图形导出为3种不同的格式，svg是2kb，而png和jpg分别为8kb、27kb。

关于 SVG 动画的历史

最早的关于 svg 动画是在上世纪90年代被开发出来，在 2003 年左右一些网页浏览器开始支持 svg 动画。但是直到最近我们还需要使用一些独立的插件或程序才可以浏览 svg 动画。

几乎所有主流的浏览器在 2011 年开始支持 svg 动画，并且在 2018 年发布了 SVG2 之后，我们才慢慢的发现更多强大且易于使用的 svg 动画软件。

使用 HTML 和 Javascript 制作 SVG 动画

从那之后网页中的 svg 动画开始慢慢取代无处不在的 GIF 动画，最近十年 GIF 动画开始慢慢消失，使用 HTML 和 Javascript 制作 SVG 动画开始变得越来越流行。

SVG 动画的使用给研发或者是设计师带来了不少的好处，目前越来越多的svg动画取代 gif 的一个原因之一是，网站的访问速度已经成为一个决定 Google 排名的重要因素之一。

如果一个网站上有过多的 gif，就会导致网站排名下降。因此，SVG这样轻量、快速、易用的格式开始变得越来越受欢迎。

SVG animator 是如何工作的？


Svg 的工作原理是需要给路径一个清晰的指导，就好比要给一个手里拿着画笔的孩子一个清晰的指引，告诉他起点、终点以及方向，按照这样的指引就可以让 svg 的路径变成一个动态效果。

通过这样的指引就可以清晰的画出直线、曲线、矩形、圆或者任意形状。

因此 SVG 路径有一个明确的起点、终点以及长度。有了这些参数，我们就能指定某个 svg 路径从 A 点到 B 点连贯起来渲染展示的动画。

Framer 的 Animator 组建是由 Benjamin den Boer 开发，用来处理让 SVG 动起来的问题，我们需要在这里点击 Copy Component，之后到 framer 项目中粘贴，再将这个组建于拖拽到项目中的 svg 资源相连接就可以使用，并且在使用过程中使用属性面板配置组建的动画设置，就可以让 svg 动起来。

这次的济州岛之行是我一个迟到了5年的约定，最早是在18年和公司团建回来之后就和老婆说下次一定要带她一起去一次，结果因为种种原因就推迟到了今年才成行。

住

我们在济州岛入住的酒店位于济州市区老衡洞和咸德海边，老衡洞是属于济州市区最便捷的地方，步行就可以到达新罗和乐天免税店，附近还有保健路步行街，有很多当地的美食店。

咸德海边可以看到号称果冻海的海景，周边有海水浴场，也是热门的旅行地之一。

济州岛的市区和我所去过的首尔、釜山、水原等大部分地区都基本一致，道路整洁干净，街边小店也很多，所以逛起来会有趣一些。

韩国的土地是私有制，土地所有者购买土地后可以根据当地法律允许的范围下进行建筑的施工修建，所以当时比起城市的公寓类型，我倒是更喜欢市区外的一些房子，都特别的有特色，这次和老婆走走停停看到了很多正在施工中的房子。

我们很多时候会在韩剧里看到男女主和家人一起生活在一起，房子是两三层的独栋，十年前在首尔建国大学附近就住过一段时间这种房子，房主是在沙特当建筑工赚钱后在建国大学附近购置的土地和房产。不过近几年大家都更喜欢住在大公司修建的高档小区，因为小区现在看起来更方便，小区内也有一些医院、学校等设施，更加的私密安全一些，现在的韩剧中的主角也是更多的住在高档公寓里。

上面提到的的两种住所很像国外的 house 和 apartment 的区别，但有一点不同的是，韩国大部分的 house 比起欧美的 house 没有那么大的前后院，韩国的 house 围墙也是离建筑比较近，围墙外就是胡同。韩国这一时期的 house 在修筑时都会有一个半地下，通常这类的半地下会当作储物空间或者租住出去，很多韩国电影中的低收入家庭会选择这种房间，像我们看到的1988种的德善一家。

济州岛的乡村房屋才是这次旅行的重点，因为济州岛是一个火山喷发而成的一个岛，所以在岛上有很多的火山石，很多乡村房屋的围墙就会就地取材将它当作院墙，院墙普遍又不高，所以穿行在小路上都能看到院落里的造景和房屋。

因为济州岛的特产是柑橘，所以路过的很多家都会在院子里种上一棵柑橘树。还有很多的庭院里会有各种主人修剪的园艺造景。

走过的时候看到好看的院子总是会忍不住拍下来。

从牛岛坐公交到涯月邑的路上就会经过很多的农田，下车后走到涯月邑的海边会经过一个村子，这个村子也有一些土地会出售。

这个村子就给人以宁静祥和的感觉，因为房主都是常住的居民，所以每个家都会被主人打理的很有特色。

在济州岛这次购物主要是集中在济州市区的免税店和七星路周边，买了一块手表和一些衣服。

一些变化

七月的西安还是比较炎热的,虽然是疫情期间,但佩戴口罩的人却不是很多.我们从钟楼附近的入口进入回民街,这里的干道横纵交错,整体上是一个非常方正的商业居民区,这里应该也是西安最大的回族聚居区,所以游客在这里还是要尊重当地居民的一些伊斯兰教宗教习俗.

陕西历史博物馆

来陕西怎么能不来历史博物馆,也可能是天气原因或者是周中的关系,陕博当天人流量并不是很大.让整个浏览体验非常舒适.

陕西历史博物馆无论是展品跨越的年代,又或者是馆藏的数量,个人游览下来是不输国博的,甚至很多国博的展品都是前期从陕博“借走未归还”“的,所以在游览过程中解说人员说,在他们这一行有一句话是:”防火防盗防国博“.

就像博物馆介绍中说明的那样,陕西长期都是中国古代历史的中心舞台.先后有14个王朝在此建都立制.尤其是周、秦、汉、唐数朝,创造了高度发达的物质文明和精神文明.

我们此次游览的 《陕西古代文明》 展是从170余万件文物藏品中遴选出的3000余件精品,所以整个展览非常有得看.

一进到博物馆就可以看到陕西的整个地貌图,是不是非常像一个跪射兵马俑?

陕西省的大部分城市都是在黄河和渭河交汇的关中平原.地图下边的汉中因为靠近川渝地区,在很多习俗上跟四川比较接近,包括饮食和方言也是非常接近.地图上边的榆林市也是一个有趣的城市,依托着丰富的自然资源,城镇整体的收入比较高,福利也是非常的好,下属的神木县甚至可以做到12年免费教育和免费医疗.教育和医疗的开销都是政府财政来负担的,这在当时推出这项制度的2009年可谓是轰动全国的一件壮举.

在西安的历史文化熏陶下,也让这里的人们展现出作为古都人民的超高文化素养,图中的小孩应该也就是刚上小学的年纪,跟着博物馆的工作人员一起聊历朝历代的历史典故,这一幕让人印象深刻.阿姨提问,小朋友回答,一问一答,皆是学问.

人面鱼纹盆

这里有很多我们曾经在历史书上才能看到的一些文物,比如图中的新石器时期仰韶文化半坡遗址中出土的人面鱼纹盆,文物级别属于一级文物.

彩绘雁鱼灯

彩绘雁鱼灯是汉代的一级铜器文物,虽然是两千多年前的文物,但是也算是非常精妙的设计,因为这盏灯在设计上可以让点灯后产生的油烟通过大雁的脖子流通道大雁的肚子中,从而避免产生的油烟在室内弥漫.也可以通过旋转灯罩来调节灯照射的方向.

独孤信印

独孤信是匈奴裔鲜卑人,他的这块独孤信印距今1500多年,印呈现八棱多面体,质地为煤精石,一共26个面,历史上虽然出土过多个多面体印章,但多达26个面的印章仅此一枚.

独孤信在历史上最为人所熟知的就是被誉为“史上第一老丈人”.

“独孤信有七个女儿，均与当时的豪门贵族联姻。史书记载，独孤信的长女是北周明敬皇后，四女是唐朝元贞皇后，七女则为隋朝文献皇后，一门出了三朝皇后，被称为‘史上第一老丈人’。”贺达炘说，“独孤信还是当时的八柱国之一，其人史书称‘美仪容’，是个美男子。著名的‘侧帽风流’的典故就是出自于他。”

不过正因为有此渊源，所以隋唐两代最高统治者身上都流淌有少数民族的血液，因此大唐盛世吸纳各方精髓、接受异族文化就没有太多的心理障碍，这也成为隋唐两代社会开放、民族平等的前提之一。

三彩骆驼载伎乐俑

这件藏品也是陕博的镇馆之宝之一,从藏品形象上可以看出,将身着汉服手持不同乐器的流动演出团刻画的非常逼真细腻.展品背后展示的是唐代当时的开放包容,会迎来世界各地的艺人来到长安.

据导游所说虽然唐三彩器物有多种颜色,但其中属蓝釉最为珍贵,蓝釉的珍贵不仅在其色泽的鲜亮明艳,更在于其中的呈色剂中的钴,唐代的钴来自于波斯,也就是当今的伊朗地区,因其稀少所以在众多唐三彩作品中才尤为珍贵.一旦某件三彩器上带有蓝彩,价格会是其他釉色的几倍甚至十几倍.

谈到唐三彩的釉色，首先了解色泽艳丽的唐三彩釉色属性。所有的釉，都是玻璃的族类。唐三彩釉是用含有铜、铁、锰、钴、锑等作釉料着色剂，并加入铅、铝作助熔剂，是一种含铅量很高的软玻璃釉。加入了铅这种对人体有害的物质,所以唐三彩不会用来做日用器皿,多是作为观赏的艺术品或者是陪葬品.制作唐三彩的无名工匠也因为长期在有毒环境中工作,导致中毒也是常有的事情.

错金杜虎符

陕博展出的虎符为左半符,年代是战国时期,距今大约2400-2600年,虎符上有9行错金铭文,字体为小篆，内容大意是：右半符掌握在国君手中，左半符在杜地军事长官手中，凡要调动50人以上的带甲兵士，杜地的左符就要与君王的右符相合，才能行动。但遇上烽火报警的紧急情况，不必会君王的右符。

铭文反映出秦以“右”为尊，秦国的军权高度集中，凡征调50人以上的带甲兵士必须经国君认可。 虎符作为中国历史上调兵遣将的凭证起源很早，信陵君“窃符救赵”的故事说明至少在战国时期虎符已经行用。而且从记载可知战国时期各国君主为把持军权都实行了类似的制度。

这件文物的出土据导游讲也是颇有意思,上世纪70年代在西安南郊一位农村少年杨东峰在农田玩耍时无意发现,在把玩了几年后,等到上面的锈迹磨光才看到里面的错金铭文,杨东峰知道是文物后主动上交，并且索要一件红卫兵穿的军服，但博物馆没有这样的军服便奖励了20多元现金。

皇后之玺

皇后之玺也是由市民捡到并上交的国宝.1968 年9月,一位 13 岁的小学生孔忠良在放学途中捡到了一块发光的白色小石头,其父看到印章后认为可能是文物便上交了国家,博物馆的一位领导为表示感谢,给了他父亲20元作为路费.

后经专家鉴定,制作印章的材料是上好的新疆和田羊脂玉.根据文物的形制和文字来看,应该是西汉时期的物品,,又根据发现的地点为汉高祖刘邦与吕后合葬墓-长陵一公里处,因此推测这块印就是吕后之物,也是汉代皇后玺的唯一实物资料,因此显得弥足珍贵.

其中专家们查阅许多古代文献，发现《汉官旧仪》上记载有：“皇后玉玺，文与帝同。皇后之玺，金螭虎纽。” 下图是目前收藏在东京国立博物馆中的皇帝信玺封泥.

这块玉玺的重要性在于,它是我国目前发现的年代最早的皇后玉玺,玉玺的主人还是历史上第一位皇后.

据导游讲关于这块玉玺还有另外一个故事,当时江青对武则天、吕后这样的女性统治者十分欣赏,也曾评论:“吕后也了不起,他对汉高祖刘邦的事业起了很大作用.”

在1974年,江青听说山西省在前些年发现“皇后之玺”是吕雉之物后,便连忙追问其下落,最后她命令陕西博物馆派人将于玺护送至北京,并迟迟不肯归还.直至四人帮被粉碎,文革结束之后于玺才回归陕西博物馆.

大唐遗宝 - 何家村窖藏出土文物展

陕博中有一个单独收费的展览颇为有趣,叫做何家村窖藏出土文物展.据导游讲解所有的我们能够看到的 1000 多件文物都是1970年出土自何家村的一个当地公安局家属楼基建工地上,当时施工的工人挖出量 2 个陶瓮,里面装有大量的金银器,据查共有 1000 多件,包含各种金银器、银铤、银板、银饼、中外钱币、宝玉珍饰和贵重药材.

郭沫若 在《出土文物二三事》中考证，这批文物“为 唐玄宗李隆基  天宝十五年 （公元756年）6月因 安禄山 之乱逃奔四川时邠王 李守礼 后人所窖藏”。他认为这批窖藏文物是邠王府中的财物，可能是在“ 安史之乱 ”时仓促埋下而后来未能挖出，所以才保留到现代。

而 陕西历史博物馆 和 北京大学 的学者考证，何家村窖藏的主人是唐代 尚书 租庸使 刘震，窖藏埋藏年代应为 唐德宗  建中四年（783年）的 泾原兵变 时，也因兵乱保存到现代。但北京大学教授荣新江认为窖藏主人是尚书租庸使刘震的说法靠不住，因为《刘无双传》中的刘震只是一个虚构的人物，缺乏其他文献的印证，窖藏宝藏应为当时的私人收藏。

从这次出土的文物无论书数量上还是种类上都是非常令人震惊的,其中定位国宝级别的文物 3 件,国家一级文物有 10 件.其中镶金兽首玛瑙杯为海内孤品,中国政府还特地禁止其出境.

鸳鸯莲瓣纹金碗

当时这种唐代的鸳鸯莲瓣纹金碗共出土了 2 件,重 391 克,每一个莲瓣上都刻有装饰图案,在最显眼的每个莲瓣内都以一种动物作主题纹饰，有狐、兔、獐、鹿、鹦鹉、鸳鸯等.这种手法常见于初、盛唐，中晚唐就不多见了，现代更是失传。

此金碗的用途是皇家酒器,两碗内壁分别墨书“九两半”、“九两三”.单从装饰上也可以看出当时的手工艺技术水平.

鎏金舞马衔杯纹银壶

同样出土于何家村的鎏金舞马衔杯纹银壶更是造型别致,采用了压模、焊接、鎏金等多种工艺,壶身两侧各有一匹翘首鼓尾、衔杯跪拜的舞马.这个形象也是印证了《明皇杂录》中记载的马衔杯祝寿的历史故事.

从器物形态上银壶模仿的是游牧民族储水的皮囊造型,这也是中原汉族与北方契丹族文化交流的一个典型物证,这也是在唐代金银器中尚属孤例.

镶金兽首玛瑙杯

这个杯的杯体是由一整块缠丝玛瑙雕琢而成的,杯流为兽首形状,并插有金塞,无论是从材质上还是工艺上都极为罕见.是迄今为止考古发现的孤例.

有一些学者认为这是一件外来器物,有可能是外邦进贡唐朝的孤品.年代不晚于 7 世纪,也有一些学者认为是唐代工匠仿栗特式来通制作的,年代为 9 世纪前期.

尽管学者们对这件玛瑙杯产地和时代的看法想去甚远,但都认为其造型来源于西方的来通,可能是艺术品或特殊的礼器,而非实用品.

西安碑林博物馆

西安碑林博物馆是在具有900多年历史的“西安碑林”基础上,利用孔庙古建筑群扩建而成的专题性博物馆,以收藏、研究和陈列历代碑石、墓志以及石刻造像为主.

现有 7 个碑石陈列室、8 个碑亭、6 个墓志廊.共展示了汉代至近代各类碑刻 4000 余方.

到碑林如果不是对内容特别了解,非常建议购买讲解服务,此次游览全程导游讲解都非常用心,了解了非常多关于碑刻的知识.

景云钟

景云钟是景云二年(711)由唐睿宗李旦(武则天四子)撰并书.钟重达 6 吨,景云钟分为上中下三段,每段分六格,除下段中格为铭文外,其余部分由动物、飞天形象装饰.铭文共 292 字,内容记述了道教(唐朝国教)的神秘玄妙和当时的人们对景云钟的夸赞.

还有我们常听到的晨钟暮鼓,其实现在西安鼓楼上的鼓是后做的,原本在鼓楼的钟最初就是这个景云钟.景云钟也是国家一级文物,属于禁止出国(境)展览文物之一.

其实我们大部分人也都听过景云钟的钟声,就是每年除夕夜中央电视台的新年钟声就是景云钟的声音.景云钟铸造的时候李白才十岁,想想我们现在听到的钟声跟李白听到的是一样的,会感觉这件事情还是挺浪漫的.

西安碑林中有非常多的著名碑石.在古代记录的四书五经,在当时相当于是我们现在的教科书.我们在这里可以看到唐代书法家欧阳询、颜真卿、柳公权,东晋书法家王羲之的墨迹刻石.

上图中的碑林二字是时任两广总督林则徐被贬新疆伊犁的途中路过西安时写下的,是因为当时虎门销烟之后,英军武力逼迫清政府签下《南京条约》后附带条件要求清廷严办林则徐,所以导致林则徐被贬,南京条约的签订导致割让了香港岛,致使中国沦为了半殖民地半封建社会.

《石台孝经》 - 唐玄宗

碑林二字下就是《石台孝经》,是唐天宝四载(745)唐玄宗李隆基作序、注释并书写.《孝经》是孔子的学生曾参记述他与孔子交谈的语录,主要内容为孝、悌之道,这块碑应该是碑林中少有的精美的碑石.

唐玄宗的隶书雍容华丽,世上人称“开元体”,此碑于北宋元祐二年(1087)移存碑林,距今已有九百余年,是碑林最早的藏品之一.

《大秦景教流行中国碑》

大秦景教流行中国碑是一座记述 景教 在 唐代 流传情况的石碑，现藏于 西安碑林博物馆 ，是中国首批 禁止出国（境）展览文物 之一。

丹麦探险家 傅里茨·何乐模 （Frits Holm）认为，大秦景教流行中国碑是了解古代中国与其他文化的交流中极为重要的石碑，当与 罗塞塔石碑 、 米沙石碑 、 阿兹特克太阳历石 并列世界四大代表性碑刻.

这里的“大秦”是中国古代对罗马帝国的称谓,“景教”是基督教聂斯脱利派(中世纪时被视为异端)传入中国后的名称.碑文记述了景教的教旨、仪式以及自唐太宗贞观九年(635)景教传入中国后一百五十年间的活动情况,并有用古叙利亚文刻写的题名.据碑文记录在当时的时代国内已经有信徒开始礼拜和过圣诞节了.

此碑为研究古代中外文化交流和基督教在中国的早期传播提供了宝贵资料.碑原立于唐长安大秦寺,明天启五年(1625)于西安城西出土,引起国内外学者重视.清光绪三十三年(1907)移存碑林

《颜氏家庙碑》

《颜氏家庙碑》在碑林博物馆也是镇馆之宝,是颜真卿在 72 岁之时撰文,为颜真卿晚年书法艺术的代表作,也是他现存最晚的作品.

隋代李静训石棺(李小孩石棺)

李静训为隋左光禄大夫、岐州刺史李敏的第四女，字小孩，生长于皇族之家，去世时年仅九岁。

李小孩的祖父李崇曾和隋文帝杨坚一起创业,是隋朝的开国功臣.父亲李敏受杨坚喜爱,从小就在皇宫里长大.而李小孩的外祖母是那个人生履历曲折的北周宣帝皇后杨丽华，她先是当了北周的皇后，然后当皇太后，隋朝建立后又成了公主，只因她是杨坚的女儿，李小孩的母亲则是杨丽华唯一的女儿宇文娥英。这样一介绍，李小孩的出生之显赫，就可见一斑.

据讲解员介绍,李静训在世时非常的受宠爱,以至于在她下葬后随葬的精美文物多达两百多件,石棺也是按照等比缩小的殿堂样式打造.目前大部分这些珍贵随葬文物在国家博物馆珍藏.在这里的也听到导游讲到了那句经典的:”防火防盗防国博“.

李小孩的墓曾经收到过盗墓贼的光顾,所以在照片中可以看到棺门有两处被撬棍破坏的痕迹.

这个墓最为著名的就是,其家人为了避免孩子的墓被盗,特意在背面的中间瓦脊上刻有“开者即死”,这就是考古历史上所称的“李小孩的诅咒”. 石棺上最狠的诅咒，却也是亲人对她最大的保护.

据导游介绍1957年在西安市西郊出土时,发现可能是年代太过久远,李小孩的遗骸已经无法与石棺分离,所以就永久的和石棺存放在一起了.

李小孩随葬品中最为著名的应该是被誉为中国考古史上发现过的最美项链.这条金项链制作技艺精湛,堪称举世无双,是中国考古史上的无价之宝,目前也存在国家博物馆.

西安城墙

来到西安一定会去的一个景点就是西安城墙,西安的城墙是现存最完整的古城墙之一,来西安之前就听说过曾经在西安城墙上还举办过马拉松,因其全长13.74公里,城墙周围有护城河环绕,西安当地人说的城内城外说的就是这里.

关于城墙的保留经历了非常多的阻力和困难.曾经在上世纪五十年代,苏联援华专家因为要建设大型军工企业的需要,主张“拆掉城墙,发展更多的道路,解决当时的交通问题”,但国内的规划专家建议将工业区放在东西郊区,且以城墙有利于防原子弹、地面冲击波为由保留了下来.

后来毛泽东在十四次最高国务会议上表示“南京、济南、长沙的城墙拆了很好,北京、开封的旧房子最好全部变成新房子”.后来的一系列会议中拆掉城墙的主张盖过了保留派,最后无奈时任山西文化局副局长的武伯纶越级向国务院上书,才获得习仲勋等人的重视,才被列入全国文物保护单位.而后，习仲勋因下令“不许拆西安城墙”等原因被革除职务，下放 河南 。武伯纶也因此被列为“反动派”、“学术权威”和“叛徒”。在之后的文革期间所幸红卫兵没有大型工程设备,且城墙过于宏大才得以保留完整.目前已经是文物了,城市建设也不再受制于城墙,我们才可以让后人能够完整的感受历史古城墙的宏伟.

其实游览城墙最快最好的方式应该就是骑车了,如果男生以最快的速度骑行一周用时大约是半小时.在途径一些区域路面时可以看到一些城墙砖上都会有姓名等信息,据说这是为了保证城墙的质量,如果哪块砖发生质量问题,都会找到相应的制砖人问责,这种责任到人的问责制度在明太祖朱元璋时期也是被广泛使用.

图片为在城墙上拍摄的西安火车站一角.西安火车站就在城墙外东北侧.西安这两个字也是从碑林中柳公权书法碑石上拓印下来的.

西安的饮食

西安的饮食一直被认为是碳水炸弹,因为特殊的地理位置等原因,西安饮食多以面食为主,西安又以其丰富的面食种类而闻名,像我们熟知的油泼面、凉皮等等.

除了面食以外当地的羊肉类饮食和西安当地小吃也比较有名,在西安的几天行程中,印象最深的还是肉夹馍和水盆羊肉.

肉夹馍推荐这家子午路张记肉夹馍,价格实惠味道正宗.一个肉夹馍就可以过足瘾.

羊肉泡馍也是来到西安必吃的一道当地特色,我们选择了回民街附近的马洪小炒泡馍,我们去的时候也有非常多的当地人,证明味道还是很正宗的.点羊肉泡馍的时候可以选择自己手掰或者是机器两种,虽然是第一次,但还是决定亲自试试掰馍,看得出大小不是非常统一.

发现很多国内的地方都有当地特色的一款饮料,在西安应该就是冰峰了.和北冰洋、宏宝莱、大窑味道很相似,都是熟悉的橙子汽水味.

水盆羊肉也是西安当地非常有特色的饮食,个人感觉口味偏淡,还是喜欢内蒙那种羊杂汤的辣汤口味.

在西安期间也实在无法全程“西安胃”,所以每天早上的酒店早餐是难得可以自选的一餐.可以吃清淡一些的蔬菜水果沙拉和炒饭.

曲江艺术博物馆

曲江艺术博物馆位于威斯汀酒店的楼下,是一家民办博物馆.凭酒店的房卡可以免费参.博物馆会定期有不同的主题,我们当时去到的时候主题是《两端-海上丝路的东方与西方》

这个展览主要是展示丝绸之路沿线文化交流为核心的展览.时代背景是18-19世纪的清朝,展品为 163 件私人收藏文物,能从展品中看到19世纪丝绸之路贸易及其带来的西方社会生活的变化.

展品多为黄金、象牙等材质的一些生活用品,展示了非常多精美的扇子,这里很多的工艺品都是中国匠人制作并外销的,虽然已经是两百年前的清朝工匠制作,但放到今天也很难有手工艺水平达到当时的标准.

海上丝绸之路上的来自世界各地的旅行家和商人共同开拓航路,他们也是中西经贸与文化交流的使者.海上丝绸之路两千多年的历史证明,和平合作、开放包容、互学互鉴、互利共赢推动了人类文明的进步.这让身处贸易战和技术封锁背景下的我有点恍惚,当下的时代究竟是进步还是倒退.

秦始皇兵马俑博物馆

秦始皇兵马俑博物馆位于山西省西安市临潼区西阳村,搭乘公共交通从西安市区到博物馆需要1小时40分钟.

秦始皇陵和兵马俑博物馆其实离的很近,关于秦陵被盗的说法比较多,也有说项羽盗过,还有说法是清朝道光年间的土匪和在之后的民国军阀刘镇华挖掘的.

但实际情况是到目前为止,有很多人认为皇陵并没有被盗,理由是市面上并没有秦始皇陵宝物流传,且随葬兵马俑保存完好、陵墓内大量水银存留、未找到大规模的盗洞.据导游讲目前为止全中国这么多皇陵中只有两座没有被盗,一座是唐高宗李治与皇后武则天的合葬墓乾陵,另外一座就是秦始皇陵.

西安作为14朝古都,除了皇陵还有数不清的太子、将军、公主的陵墓,所以陕西的盗墓高手非常的多,导游介绍其实国内最早的一批文物工作者都是收编自民间比较厉害的盗墓高手.不开玩笑的说上个世纪七八十年代,所有的全中国最厉害的盗墓贼都在陕西.

秦陵兵马俑被称为世界公认七大奇迹之外的第八大奇迹,兵马俑的发现还是因为一次偶然,1974年3月29日陕西大旱,临潼县的村民杨志发在挖井打水时意外发现兵马俑的碎片,之后4月25日任职临潼县文化馆的赵康民接到村民电话说发现了残破陶俑,后鉴定为兵马俑.此后开始了兵马俑的挖掘修复工作.所以从考古意义上来说,秦俑的真正发现者是赵康民.

最初兵马俑被发现时并不是我们所看到的灰色,当时兵马俑的衣着、武器的颜色十分的鲜艳,不敢相信经历了2000多年还能保持的如此之好,不过人们惊讶的发现没过几分钟颜色就会被氧化,剥落成陶土色,所以后来很多挖出来的陶俑为了更好的保护起来都用土回填回去了.

导游通过手机照片给我们展示了兵马俑刚出土时的真实颜色,根据维基百科记载兵马俑的原漆有紫黑红绿等十种颜色,但通常出后后与空气接触不到五分钟就氧化脱落了.目前据导游讲世界上最先进的彩绘保护技术是一家德国的公司,它们正在提供技术和中国的团队一起尝试将兵马俑的颜色保留下来,在 2015 年日本NHK特辑节目中播出的内容显示,目前经过处理的残留颜色兵马俑,已经可以暴露于空气中展示.

我们在兵马俑博物馆看到的其实只是成千上万个当中的一部分,还有很多是深埋在土里还没被挖掘,据导游讲述,会在技术稳定且成熟时再将鲜活的兵马俑展示给公众,我们要和技术赛跑去尽可能早的完美修复,也要给后人留下充足的文物供他们去研究.

不知道看照片能否辨认,兵马俑坑中的土不同于陕西其他地区的黄土,这里的土质颜色呈红褐色,后来经过专家鉴定后发现这里的土不含有植物所需要的矿物质和微量元素,这里的土都是经过人工的炒制之后的熟土.

据导游讲解兵马俑的陶片经过鉴定不但含有这种炒制的熟土,还含有糯米汁的成分和硅酸盐(水泥中含70%硅酸盐)的成分. 这里就可以知道水泥虽然是近代的产物,但早在两千多年前秦朝的匠人已经如此大规模的使用了.

在修复的过程中发现兵马俑的制作是分件式制作的,兵马俑的头、手、还有兵器都是可以活动的.在一些兵马俑的隐秘部位比如一些头部和手部还会会写上一个名字还有时间.这样都可以找到生产者,有点像生产责任制,监工是谁、工匠是谁都可以对应到.如果哪个部分没有做好,直接找到对应的人去砍头.

还有一个有趣的发现是兵马俑的小腿往下都是实心的,小腿往上都是空心的,这么做的原因是可以让重心尽可能靠下,容易站立得住设计非常的合理.截止目前还有615处随葬坑没有被打开,也就是说他们已经在兵马俑坑里面站了两千多年.

一些兵马俑会用塑料膜包起来,防止其过快氧化.

图片中可以看到博物馆一角的开放式修复室,导游讲虽然修复文物所需的设备非常昂贵,但得益于兵马俑博物馆约 8 亿人民币的门票收入(2017年),还是可以保障文物修复等各项开支.目前国内所有的博物馆中,兵马俑博物馆是少数几个能盈利的博物馆之一.

正面拍摄的兵马俑坑,可以感受到兵马俑博物馆规模之大.

这张图片展示的是整个兵马俑坑中出土的少数军官佣之一,截至目前只出土了7件,是级别最高的陶俑.这类佣的特点是”头戴双尾鹖冠、身穿双重长襦、脚穿方口翘尖履“.着盔甲者,甲为彩色鱼鳞甲,甲片小,甲衣短,甲的周边饰精美的花纹图案,双肩及前后胸部有彩带系绾的花结.

今年五一期间借着假期和年假,陪老婆一起逛了香港、深圳、珠海、澳门、顺德和广州. 体会了不同城市之间的人文风景差异.实在是一趟难忘的行程,很久之前就希望能够去一次香港和澳门,这次的行程算是圆了一个心愿.通过这次的旅行,我们的旅行足迹又多了几座城市.

这篇文章主要是记录我们在香港的一些旅行经历,分享我们眼中看到的香港.

因为逛的地方比较多,但相对距离没有很远,所以行程中大部分都是步行,平均单日步数都不下2万步,妥妥的一顿暴走 city walk.

出行前的准备

  1. 港澳通行证:出行前最重要的应该是港澳通行证了,我们是在海淀中关村西区派出所完成的申请和签注,需要提前预约时间,之后带着身份证和居住证前往就可以.办理时间很快,全程不到一个小时,费用一人100+.
    需要注意的是如果是临近出行,可以直接在派出所完成签注,这样就免去了通关前排队签注的时间.
  2. 港币: 因为香港的一些店是不支持大陆的移动支付,只收取现金,所以我们提前在中国银行的小程序预约了购汇业务,两个人一共换了1000港币,按照当时的汇率应该是8.8左右.
    澳门的店家也是收取港币的,所以不用单独换澳门元.直接使用港币即可,港币和澳门元之间的汇率差可以忽略不计.如果是没有港币,一些店家也可以收取人民币,不过是1:1的汇率收取,如果是这样就会有点亏.
  3. 攻略: 出行之前的很多攻略都是在小红书上看到的,我会提前把想要去的地方在地图上标记出来,再了解当地的一些交通出行就可以轻松的自由行了.
    这次的出行原则就是不去游客扎堆的网红餐厅、景点.更多的时间放在感受当地的一些人文上.不想把宝贵的时间浪费在排队上.
  4. 住宿:香港的住宿我是在 Airbnb 上提前预定的,位于尖沙咀弥敦道附近的公寓.香港的住宿是比较贵,幸好提前预定,不然会话很多冤枉钱.

出发

出发时北京下了很大的雨,导致飞机延误了四个小时才起飞,出发大厅里挤满了等待航班起飞的乘客.这让我们提前感受到了节日假期人挤人的感觉.

不过与之形成对比的是,首都机场的商业还是肉眼可见的凋敝,没想到疫情过去一段时间之后,一些店铺还是没有开业,这跟我们在五道口和中关村商场逛街时看到的感觉是一样的.可能恢复疫情前还需要一些时间.

作为一个北方人,香港一直给我的感觉是是一个很遥远的城市.直到这次出行我才了解原来往来香港已经如此的便利.

我们从北京出发,飞机到达广州飞行时间在3小时左右.到达广州后,高铁1小时可以直达西九龙站,出高铁站步行就可以达到维多利亚港.如果是选择深圳入关会更快,高铁18分钟,也可以选择地铁抵达香港,乘坐地铁的时间比我从中关村去大望路办公室时间还短.

目的地城市也是一直阴天.虽然出发前看了香港的天气,也很担心会因为天气的原因导致出行受阻.

庆幸的是,在香港的 2 天,天气一直很好.当日的天气非常适合在街上散步闲逛.一出高铁站就感受到有种湿热的海风吹在脸上的感觉.气温比在北京要高出近10摄氏度左右.

九龙半岛

出行的绝大部分目的地都在九龙半岛,所以我们来到香港的第一印象就是从九龙来的.老婆说知道香港老,没想到楼都这么老.给人很强的时代感,抬头不经意间感觉穿越到了二三十年前.路边的楼确实都是很有年代感,很少大面积的住宅小区,看惯了国内的一些新城,确实会有感觉衰败的感觉.

上面的照片是在去中银香港土瓜湾营业厅的路上拍到的居民楼,从照片中就能体会到当时的感觉了.照片中前后新旧居民区的对比还是很强烈的,前面这种楼宇的外立面在国内的老城区也经常能见到.

照片是随手在土瓜湾街边的房屋中介拍到的房产价格,粗略算了一下图中一套795尺(88平米)的房子售价2680万港币,算下来一平米合人民币27万左右.着实是感受到了亚洲最高的房价,这还是九龙,如果是港岛中环附近或者是太平山的房子都不知道能到多少.

漫步穿梭在香港的街巷中,我们想到的都是首尔,两座城市给人的感觉都是那种小街道,人流比较大,而且楼和楼之间贴的比较近.不同的是两座城市的招牌文字,以及路上的行人.

我是非常喜欢这种城市街道,大片的商业街区都很发达,有点区别于北京的商场购物区的体验,这里的商业很多都是临街商业,各式各样的餐厅、小店都非常有的逛.四通八达的小路走到哪里都会有新的体验.

在路上也时常能出现一些小意外,原本计划前往的一家茶餐厅没有营业,所以我们索性就近找了另外一家,到达之后才知道这也是一家网红餐厅,但好在来的早没有排很久的队.

这家华嫂冰室是香港非常有名的一家传统茶餐厅,经营本地地道的港式点心、汤和煲仔饭.最早是由广东来港的华侨于1953年创办.

虽然是一家网红餐厅,但价格不贵,两个人的早餐花费109港币.点了一份菠萝包、猪排滨城喇沙汤、冻奶茶和奶油西多士. 老婆很喜欢喇沙汤的味道,是一种很浓郁的香料的味道,跟咖喱很像.菠萝包味道也很赞,面包口感酥脆柔软.

等到我们出来的时候餐厅才真正开始排起了长队,随意瞥见门口的招工启示发现这里的服务员的薪资情况:兼职时薪75港币/时,煮面的服务员21000-22000每月.

所以对比来看今天我们两个人的早饭相当于普通服务员1.5h的收入.由此可以大概了解香港的薪资购买力.

华嫂冰室附近还有一个非常有名的打卡点就是重庆大厦,大家知道这栋楼是因为王家卫的电影重庆森林.这栋重庆大厦还被称作”联合国大厦“,它本是一座大型外籍人士宿舍和商业大厦,建于1970年.许多来自世界各地的外籍人士都生活在这里.

比较遗憾的是我们当时并没有太留意重庆大厦,散步时路过都没有拍照.重庆大厦离吃早餐的华嫂冰室直线距路可能都不到200米.

维多利亚港

维港应该是每个来港的朋友都会选择打卡的地方,给人感觉和外滩很像.都是在一侧远眺对面的高楼的感觉.

维多利亚港是一个天然良港,虽然肉眼觉得不是很宽,但却是一个深水港,港口的交通也是非常的繁忙活跃.他是一个非常重要的国际航运枢纽.

九龙尖沙咀海滨的星光大道是一个致敬香港电影工业和明星的地方,有非常多的游客前来打卡,这里类似好莱坞的星光大道.

星光大道的设计灵感是源于电影胶片,沿着海滨的步行道上铺设着黑色的地面,模拟电影胶片的形状.

步行道两侧有许多铜像和雕塑，展示了香港电影工业的巨星和重要人物的形象。游客可以在这里找到像李小龙（Bruce Lee）、周润发（Chow Yun-fat）、张曼玉（Maggie Cheung）等众多著名明星的雕像。

天星小轮

天星小轮是香港的一个标志性的公共交通服务,提供乘客穿越维多利亚港的船运服务. 天星小轮的历史可以追溯到1888年,起初是有苏格兰伤人创办的一家私人轮渡公司.

天星小轮采用了经典的双层设计,内部的座椅也是非常传统的木制座椅.提供了非常好的观赏视野,整体航行时间不长,很多人都选择吹着海风感受维多利亚港的夜景.

天星小轮周中票价5港币,周末6港币.节假日可能排队会比较久,如果有八达通卡可以直接刷卡,可以走刷卡通道,排队时间会少一些.

中环

香港中环是香港岛上一个核心商业和金融区,也是香港的中央商务区.这里的整体感觉就会和九龙岛差异比较大.在这里就可以体验最繁华的香港.

中环还有非常多著名的高端品牌的旗舰店、购物中心和跨国银行以及集团总部. 标志性地标建筑有—中银大厦和香港汇丰银行大厦.他们都是非常著名的建筑设计作品.

非常喜欢这里的一点是在中环有非常多的室内电梯和步行道.不同于重庆的将汽车抬到到高架,这里是将行人抬到“高架”.这里的高架就是每栋楼之间的走廊.

这样做就可以在繁忙拥挤的中环,将人车进行有效的分流,而且行人可以更好的享受高视野带来的景色,部分封闭的走廊还可以享受到商场里带出的冷气,这对于行人非常的友好.

对于在YUST上学的人来讲,这种设施就是我们熟悉的连接栋,无论外面刮风下雨都不会影响我们的出行,中环的行人交通应该是我目前为止见到过最便捷的方式,行人可以选择临街小路,楼栋之间的走廊,还有地下的地铁,非常的有层次.

叮叮车🚎

港岛的叮叮车🚎也是一个特色交通方式,在九龙是体验不到的,在国内也有一些城市保留有这种有轨电车,比如大连也有非常有特色的电车.但不同于大连的电车,香港的叮叮车分为上下两层,乘客从后门上车,可以选择视野更宽阔的上层乘坐.

叮叮车的历史最早可以追溯到1904年,至今已有一百多年的历史.最初的时候是由马车作为动力进行运营.

叮叮车的行驶速度很慢,且经过的地方多为香港岛的中环、湾仔和北角等地.大部分都是香港繁忙的街道和热闹的市区,所以为游客提供了一种慢节奏的城市观光方式.我们也可以借着叮叮车游览香港的城市风光,感受独特的城市氛围.

叮叮车的票价也非常的便宜,只需要3港币,在下车时投入硬币或者刷八达通卡即可.

虽然来香港之前一直觉得香港物价高,但是实际到达之后发现一些费用还是很便宜的,比如叮叮车、天星小轮、还有中环的摩天轮,价格都非常的划算.

关于住宿

香港的住宿是出发前在 Airbnb 提前订好的,位于尖沙咀弥敦道附近的公寓房间.比起国内的同类型公寓价格贵了一倍左右,但在香港这个价格已经是很划算了,价格和在北京住一个连锁酒店的价格一样.

出发前就知道香港的住宿贵且面积通常比较小,但实际看到后还是有一些吃惊. 这应该是目前为止住过的公寓中最袖珍的了,面积差不多4-5平米左右,还是算上了卫生间的面积.

公寓是一间大的公寓拆分为多个房间,中间是走廊,有点像学生公寓这种.类似的房间可以参考韩国的考试院的房间,里面的摆设只有一张床、一个床头柜、一个挂墙电视和空调.

不过卫生还是很干净,能够在五一假期抢到一间这样的房间已经很满意了,毕竟价格也不算太贵.而且还是在尖沙咀这样的位置.

如果来到香港住宿并且屋内安装了有线电视的话,还是非常建议看一下的.里面的很多节目都非常的有特色,就是连广告的设计也是很有港式风格.还有一个就是香港的新闻,会发现和内地还是有很大的不同,报道和数据都是非常的写实.老婆甚至还说如果在北京也能在家收到TVB和明珠台就好了.

关于货币

来港前我们就在家附近的中国银行提前预约了购汇,通过微信公众号可以提前预约,看了网上游客的分享,一般两个人游玩两三天基本1000港币就够了.1000港币折合人民币880左右.

在香港一些小店会收取纸币,很多地方已经支持支付宝付款,根据实时汇率进行结算.如果能电子支付还是尽可能的选择电子支付.

在国内预约换港币也是受到外汇管制的监管,算在每人每年5万美金额度之内.

如何提升办公效率

  • 工作时间远离手机

工作时间远离手机尤其是短视频、社交媒体尤为重要，这些应用在我看来都是“时间杀手”，规律且有节制的使用手机是提升工作效率非常必要的手段。就算是在之前的工作经历中设计移动端应用时，我也会尽可能不使用自己的手机作为测试机，而是使用各种不同型号、未安装任何无关应用的手机作为测试机。

慢慢的通过远离社交媒体和短视频应用后会感觉时间开始真正属于自己，做事情时也会更加专注。

  • 善用邮件订阅

因为远程办公有太多异步沟通的场景，所以邮件是必不可少的。因为团队使用 极狐GitLab 的 issue 作为项目管理的工具，所以通常每天都会收到很多的工作邮件，有讨论的内容，项目的状态，是否需要设计跟进等等信息。

因为公司会使用 gmail 作为团队邮箱，我为了避免特殊情况下漏收邮件，会将 gmail 邮件自动设置转发到我的网易邮件客户端，这样就不用经常在浏览器中找到 gmail 的 tab，会让工作更加的高效。

其次非常有用的一点是你可以在议题列表中单独关注某些标签，比如我会将::Usability:: 、::UX::、::Frontend:: 设置为关注，每当有团队成员或者社区贡献者提交议题并附上以上标签时就会邮件通知我议题内容，我就可以第一时间跟进并关注。

  • 及时沟通

这是一个非常重要的一点，在远程办公中最重要的一点是及时沟通。每当团队中一起讨论每个 milestone 中的一些 good/bad 时，很多同事都会反应希望工作中不要有太多等待时间，比如有一些贡献的代码是需要跨时区的研发来review或者确认，这就有可能会产生意料之外的时间损耗，原本计划2天完成的任务有可能就会拖延。

作为设计师我会有一些稳定的研发同事，我们会经常在任务进行中沟通，他们都有着非常棒的工作习惯。会在遇到设计相关的议题时及时@设计师加入，在设计评审时非常了解我们的设计系统，并在文档中快速找到相应的说明，也会在同步或异步沟通中快速有效的解决问题，这让我在工作中非常的安心，我知道我遇到的问题或者提交的bug他们都会非常靠谱的帮忙解决。

在远程办公的环境下不回复信息或经常性的延迟回复信息，会容易让合作的伙伴很难产生信任感，通常的做法是第一时间沟通事情的优先级或大致的解决方案和deadline，让对方知道你有在关注这件事情。如果真的非常困难或者时间排不开，我们就需要找到我们的manager沟通，我们不是超人，不可能面面俱到的处理好每件事情。

承认自己不可能事事做到100分会让自己的工作压力瞬间减少很多。同样对伙伴的信任和理解也会让合作变得更加愉快。

沟通的愉快且顺利会让我感觉完成任务就像是在通关打游戏，发现的每一个待优化点都像是地图上的一块拼图，非常的有成就感。

  • 使用勿扰模式

在晚上十点之后将手机设为勿扰模式也被我列入到提升办公效率的一项，通常我不会在深夜完成重要的工作，劳累了一天之后我希望身体得到充分的休息，这也是提升办公效率的方式之一,精力管理非常重要。

因为有一些议题提交到 GitLab 之后，会由其他时区的同事在讨论回复，所以往往会在半夜收到邮件提醒，如果睡前大脑过度活跃就会影响睡眠质量，工作和生活的边界感需要我们刻意去设定并遵循。

  • 添加待办事项

在团队中使用的项目管理工具中有一个【添加待办事项】的功能，在议题页面右上角点击待办事项，就可以将议题加入到一个todo list，这样就不用再担心遗漏议题了，根据优先级一件一件的完成也会极大的提升工作幸福感。

设计管理是 极狐GitLab 议题功能内的一个模块，在这里设计师可以上传议题相关的设计资产，包括线框图、原型图等。

如何使用设计管理

首先要求启用 Large File Storage（LFS）：

  • 对于自助管理实例，管理员必须全局启用LFS。
  • 对于使用极狐GitLab SaaS的用户是默认开启的。
  • 必须为项目本身启用 LFS。如果全局启用，默认情况下所有项目都启用 LFS。 要在项目级别启用 LFS，请导航到项目的 设置 > 通用，展开 可见性，项目功能，权限 并启用 Git 大文件存储 (LFS)。

目前设计管理支持上传的文件类型有png、jpg、jpeg、GIF、bmp、tiff、ico、webp 或 svg 格式的文件。

用户可以先打开 jihulab.com 创建一个项目，之后进入左侧面板中的议题列表页面新建一个议题，创建完成后点击议题界面中的点击上传，或将图片拖拽至虚线框内完成上传。

上传完成的设计文件会在议题页面显示，我们在上传同名文件时会生成一个新的版本，但如果是同名文件但内容没有更改过时则会跳过且不会更新版本。

所有拥有议题访问权限的用户都可以看到上传的文件，用户可以点击图像进行放大，上传多张设计图时也可以使用键盘的左/右键来切换。

如果用户希望对图片进行评审，可以通过鼠标点击图片中需要评论的部分来进行留言，这与我们在Figma或是同类产品的操作逻辑类似。

在设计评审时我们可以@项目中的相关人员，与他们进行交流和反馈，如果一些交流和反馈得到了解决，也可以将这些评审标记为已解决。

通过点击右上角的【添加待办事项】也可以将设计评审添加到自己的todo list，避免因为议题过多而遗漏。

我们在设计管理中所生成的用户活动事件都会由极狐GitLab跟踪并显示在用户配置文件、群组和项目活动页面。

什么是SVGA

SVGA是一种同时兼容 iOS / Android / Flutter / Web 多个系统平台的动画格式。

通过官网文章中的介绍简单理解他的实现方式,就是在设计软件中将所需要的元素导出,之后SVGA会生成一个脚本,记录着这些元素的移动和变化,并将这些在各个系统平台拼合出动画.

他的优点是

  1. 相比GIF颜色效果好(8位 VS 32位).
  2. 播放资源占用小(内存和CPU).
  3. 动画文件小(同一个文件PNG序列为1.8M,SVGA为150kb).
  4. 动画过程没有差值计算,每一帧都是丝毫不差的还原设计.

实现过程

设计的需求其实很简单,市面上也有很多优秀的拆红包的动效.

设计的过程先是在figma里完成静态的设计,在考虑完它的动画效果后把资源导入到AE里面继续完成动效的制作.

在AE里面主要是使用了3D图层,模拟红包在Y轴上翻转效果,以及红包上下分开的位移.整体的效果比较简单.在一些关键帧辅助上加入了缓动,让动画不再那么生硬.

遇到的问题

第一次上手SVGA的过程中也遇到了一些问题,但好在靠着网上的资源都找到了解决办法.

  1. 导入到AE的设计资源为PDF格式,导致整体文件过大,无法正常播放
    可以在ae里面替换压缩过的PNG图片,内存占用会大大降低.(项目实测:1809M→1.54M)
  2. 3D旋转没有被SVGA生成
    发现SVGA会有一些特效是暂时不支持的,比如:
    粒子效果
    羽化
    遮罩
    颜色替换
    3D图层效果
    操控点工具
    表达式
    流体效果
    高斯模糊
    描边
    位移曲率
  3. 文本图层不会生成image key

因为开发同学要引用红包最后展示的金额,所以设计需要告诉开发金额现实的具体位置.但是在AE里面使用Text Layer标注的位置并没有生成image key供开发定位位置.

所以最后再合成里新添加了一块矩形元素,大小和位置和原本的文字区域重合.之后重新生成了SVGA文件,并把矩形所在的位置作为image key供开发同学定位.

复杂动画解决方法

因为当前的SVGA是不支持较为复杂的效果(比如3D图层),但好在因为SVGA是支持PNG序列的,所以就可以将整个动画先转出成PNG序列,然后用AE再打开这些PNG序列,最后使用SVGA Converter_AE工具生成SVGA文件交付开发.

因为每个PNG序列的透明像素都会造成资源过大,所以要把除红包以外的空白像素都截掉再导入到AE文件中.

导出完成后还要使用压缩工具将每个png资源压缩到尽可能的小.

最后可以在AE的插件SVGA Converter_AE中预览到动画效果,并且标注了内存的占用大小.
